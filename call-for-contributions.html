Submission Guidelines

Format Requirements

  • Use the IEEE ICRA conference template
  • Submit as PDF format
  • Include all figures and tables
  • Ensure proper citation format

Review Process

  • Double-blind review process
  • Each submission reviewed by 2-3 reviewers
  • Reviews focus on novelty, technical quality, and relevance
  • Authors will receive detailed feedback

Presentation Format

  • Accepted papers: 10-12 minute presentation + 3-5 minute Q&A
  • Extended abstracts: 5-7 minute presentation + 2-3 minute Q&A
  • Demos: 5-10 minute live demonstration
